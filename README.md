Keeping up with the aesthetic, I made a world Clock Split-Flap Solari board. 

To run on your machine simply copy the repository or download the 3 files: index.html docker-compose.yml and Dockerfile into the same folder. 

Then run " docker compose up -d --build " now your World Clock dashboard is running on your machine at port 8056 http://localhost:8056

This app was a simple iteration of html and made mobile friendly, ready to use on fullscreen either on laptops, tablets or mobile.

It has a half top with 6 static cities, and a bottom with randon cities every 10 seconds

For those who have friends around the globe!
